# VolleyApp (Firebase Versija)

Tai yra realaus laiko tinklinio registracijos sistema, sukurta su **Firebase**.

## Privalumai

- **Momentinis atsinaujinimas**: Kai draugas užsiregistruoja, jo vardas atsiranda ekrane akimirksniu (nereikia perkrauti).
- **Jokio serverio**: Nereikia pirkti hostingo ar mokėti už PHP serverį.
- **Saugumas**: Viską valdo Google infrastruktūra.

## Kaip pradėti?

1. Atidarykite `SETUP_CHECKLIST.md` failą ir sekite instrukcijas.
2. Jums reikės pakeisti `firebaseConfig` duomenis faile `index.html`.
3. Paleiskite komandą `firebase deploy` ir viskas veiks!

## Technologijos

- **Frontend**: HTML5 + Tailwind CSS
- **Backend**: Firebase (Firestore, Auth, Hosting)
